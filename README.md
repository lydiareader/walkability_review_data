# Systematic Review for Walkability and Physical Activity in Children and Adolescents
This repository houses the data and analysis code that was used for "Systematic review of the correlates of walkability and physical activity among children and adolescents". 

**coded_studies.csv** contains the results of each study coded along the described walkability schema. 

**cleaning_and_high_level.ipynb** is how the coded data was cleaned for analysis.

**analysis.ipynb** contains the code to conduct the analysis.
